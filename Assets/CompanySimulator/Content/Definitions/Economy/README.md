# Economy Definitions

Bu klasör global ekonomi ayarlarının asset olarak tutulması için ayrılmıştır.

Planlanan asset tipleri:
- `EconomyBalanceDefinition`
- `BudgetResponseCurveDefinition`
- `EconomySetupDefinition`
- `ProjectExecutionDefinition`

Bu assetler kodu değiştirmeden ekonomi balansı yapmayı kolaylaştırır.

Örnek içerik üretmek için Unity içinden `Company Simulator/Generate/Sample Economy Content` menüsü kullanılabilir.
